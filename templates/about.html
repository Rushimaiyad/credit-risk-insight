About the Project

Project Overview

"Credit Risk Insight" is an academic demonstration predicting loan default probability using modern Machine Learning techniques. By analyzing borrower profiles—including income, loan size, credit score, and employment background—the system outputs a data-driven risk score.

Technology Stack

  • Backend: Python, Flask
  • Machine Learning: Scikit-Learn (Logistic Regression, Standardization)
  • Frontend: HTML, CSS, Vanilla JS
  • Deployment: Render (via Gunicorn)

Machine Learning Model

The core engine utilizes a Logistic Regression classifier for binomial classification (Default vs. Paid). Input features are systematically normalized using StandardScaler to ensure balanced weighted importance. The model uses predict_proba() to return a continuous risk percentage rather than a hard binary classification, allowing for tiered risk profiling.
